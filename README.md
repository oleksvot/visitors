# Traffic & Visitors Bot

Простий телеграм бот, реалізований на Python з використанням aiogram та sanic.
Основне призначення – збір даних по використанню трафіку з декількох VPS та надсилання цих даних у телеграм.
Також можливе відображення поденної статистики на веб-сторінці.
Рахується загальний відсоток від денного ліміту для Oracle Cloud Free (300 ГБ).
Також може бути використаний для відстеження відвідувачів сайту (http-заголовки), та виконання запитів до служби whois.
